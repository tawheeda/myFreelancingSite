Zombie Arena

This is a text-based zombie survival game in C# called Zombie Arena. The player battles waves of randomly generated zombies with increasing difficulty. On each wave, the player can choose to attack, use a healing potion, or flee (with a penalty). After surviving a wave, the player may receive random rewards like health, potions, or increased attack power. The game continues until the player's health drops to zero, at which point their final wave count is displayed.

Scrabble

This is a simplified two-player Scrabble word game in C#. Players take turns entering words, and the program checks if each word is valid based on a predefined dictionary. If valid, it calculates the word's score using classic Scrabble letter values and keeps track of each player's total. The game continues until a player types "EXIT", after which the final scores are displayed along with the winner.
